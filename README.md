## ğŸš€ Quick Start

### Prerequisites
- **Node.js 18+** (LTS recommended)
- **npm**, **yarn**, **pnpm**, or **bun**

### Installation

```bash
# Clone repository
git clone https://github.com/Rene-Kuhm/rene-kuhm-portfolio.git
cd rene-kuhm-portfolio

# Install dependencies
npm install  # or yarn/pnpm/bun install
```
